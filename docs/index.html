Features

Everything you need.
Nothing you don't.

Built on the same editor engine as VS Code, packaged as a focused desktop app for writers and developers.

✍️
Monaco Editor
Full VS Code editor engine with autocomplete, syntax highlighting, multi-cursor, and regex search.
👁️
Live Preview
Real-time Markdown rendering with auto-scroll sync, GFM tables, and code block highlighting.
📐
KaTeX Math
Full LaTeX math rendering — inline $...$ and block $$...$$ equations.
🪟
Flexible Views
Toggle between editor-only, preview-only, and split view instantly. Resizable divider.
📑
Multi-Tab Workspace
Open multiple files side by side. Tabs persist state including scroll position and cursor.
💾
Auto-Save & Recovery
Configurable auto-save interval with crash recovery backups. Never lose your work.
📁
Folder Workspace
Open entire folders as workspaces. Sidebar file explorer with create, rename, and delete.
⌨️
Command Palette
Cmd+K for everything. Search and execute any action without touching the mouse.
🎨
Light / Dark / System
Three themes with system auto-follow. Accent color customization for a personal touch.
Development

Get started in seconds.

Clone, install, and run. Electron + React + TypeScript — no magic, just clean code.

# Clone and install
git clone https://github.com/hongmacho/machodown.git
cd machodown
npm install

# Start development
npm run dev

# Run tests
npm test               # vitest
npm run typecheck       # tsc --noEmit
npm run lint            # eslint
