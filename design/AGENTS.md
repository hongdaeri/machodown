# design

## Purpose
UI 프로토타입 및 디자인 참고용 파일 디렉토리. 프로덕션 코드가 아닌 레이아웃/스타일 탐색용 파일들로, 실제 앱 빌드에 포함되지 않는다.

## For AI Agents

### Working In This Directory
- 이 디렉토리의 파일들은 **프로덕션 빌드에 포함되지 않음** — 참고 전용
- JSX 프로토타입과 CSS 파일이 포함되어 있으나 `src/renderer/`와 독립적
- 실제 컴포넌트 구현 시 이 디렉토리의 디자인을 참고해 `src/renderer/src/components/`에 구현
